# About Me:
🔭 I'm currently working on: Building my developer portfolio at kaiserkhan.com and a headless commerce storefront using Next.js with the Shopify and BigCommerce Storefront APIs.<br>👯 I'm looking to collaborate on: Headless and composable commerce projects, and open-source tooling for BigCommerce and Shopify developers.<br>🤝 I'm looking for help with: Great patterns and tooling recommendations for modern React and Node in a headless commerce context.<br>🌱 I'm currently learning: Node.js, Express, React, headless commerce (Hydrogen and Next.js), and PostgreSQL.<br>💬 Ask me about: Custom BigCommerce builds, Stencil themes and widgets, e-commerce REST and GraphQL integrations, PIM and inventory systems, Salesforce ERP sync, and Laravel.<br>⚡ Fun fact: I built a hospital management system in Laravel that took two hospitals fully paperless, and a budgeting platform that traces spending across operations in 11 countries.
